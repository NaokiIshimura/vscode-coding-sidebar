# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in AI Coding Panel.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+A` / `Ctrl+Shift+A` | Focus AI Coding Panel |

## Panel Shortcuts (when panel is focused)

| Shortcut | Action |
|----------|--------|
| `Cmd+S` / `Ctrl+S` | Create new Plan (directory with markdown file) |
| `Cmd+M` / `Ctrl+M` | Create new Markdown file |

## Editor Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+R` / `Ctrl+R` | Run command in Terminal |

## Quick Actions

### Plans View
- Click directory to navigate into it
- Click ".." to go to parent directory
- Right-click for context menu actions

### Editor View
- Save button or auto-save when switching files
- Run button to execute command
- Plan button for implementation planning

### Terminal View
- Click "+" to add new tab
- Click "x" to close tab
- Click links to open URLs or files

## Tips

1. **Quick Panel Access**: Use `Cmd+Shift+A` to quickly focus the panel from anywhere
2. **Fast Plan Creation**: Press `Cmd+S` while in the panel to create a new plan
3. **Rapid Iteration**: Use `Cmd+R` to quickly send commands to Terminal
